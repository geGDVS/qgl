# 全攻略测试网站（SEWT）

这是一个用于考察用户对高考英语词汇全攻略中中文所对应英文单词、短语和句子掌握情况的网站。

## 功能特点
1. **多样化考察**：可考察单词、短语和句子，在考察短语和句子时会显示重点考察单词。
3. **切换题目**：点击左上角可选择题目集。

## 使用方法
1. 打开 `index.html` 文件，即可开始测试（或在线访问 [https://qgl.pages.dev/](https://qgl.pages.dev/) ）。
2. 查看显示的中文内容和考察类型。
3. 点击“SHOW ANS”按钮查看正确答案。
4. 点击“NEXT”按钮切换到下一个题目。
6. 点击左上角可选择题目集。
